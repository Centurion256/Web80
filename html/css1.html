About CSS

---------

This is the second part of this tutorial, which will be primarily focused on styling the elements.


Cascading Style Sheets (CSS) is a stylesheet language used to describe the presentation of a document written in HTML. CSS describes how elements should be rendered on screen, on paper, in speech, or on other media. For example, CSS can be used to alter the font, color, size, and spacing of your content, split it into multiple columns, or add animations and other decorative features.


Remember the class and id elements that we introduced in HTML? Most of the time, they are used in cojunction with a corresponding CSS selector, which specifies the look of that element. More on that in the following articles.
