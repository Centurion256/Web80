HTML: Attributes

----------------

In the previous section we mentioned that elements can have attributes and that these attributes can either somehow identify the element or set some of it's properties.


Consider the example from the previous page. If we add an attribute class to it, and set it to some value, we'll get this:

In this case, class attribute allows you to give an element an identifying name, that can be used to target said element by CSS.


All HTML attributes have the following properties:

  • All HTML elements can have attributes
  • Attributes provide additional information about an element
  • Attributes are always specified in the start tag
  • An attribute should have a space between it and the element name or the previous attribute


While most attributes come in attribute=value pairs, some attributes can only have the values true or false, therefore if their name is present, their value is considered to be true, otherwise it is false. Such attributes are called Boolean attributes.


There is another distinction between the atributes: global and element-specific.


Global attributes are attributes common to all HTML elements; they can be used on all elements, though they may have no effect on some elements. class is one example of such attributes, it can be present on any element, and use used to identify groups of elements. Another such element is id. It serves a similar purpose as class, but there must exist only one unique element associated with one unique id, while multiple elements can have the same class.


Non-global or element-specific attributes are attributes that can be present only on specific set of attributes, and can have different meanings in each of them. They also tend to have some reserved values associated with them. An example would be the href attribute of the <a> element.
Element <a> or the "anchor"element is used to create a link to another document. The content of the element is it's content representation(most commonly text), while the href attribute is used to actually specify the URL.
